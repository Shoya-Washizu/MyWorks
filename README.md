## card_img_maker_out

自主制作ボードゲームにおける，画像データ以外の作成環境とコード．

中心コードであるmain.pyとcardmake.pyのみを入れてある．

実際の開発環境のフォルダ構成（の一部）は次のようになっている．

    codes /main.py       ...実行することでカード画像が表示される．
      /cardmake.py       ...画像作成部分．
      /carddata.xlsx    ...カードデータを入力してある）
    materials/costs/
          /flames/
          /pictures/     ...カードの素材が入っているフォルダ）
